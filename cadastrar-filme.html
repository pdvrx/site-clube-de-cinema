Como cadastrar

O nosso site utiliza a base de dados fornecida pelo site The Movie Database. Para fazer o cadastro de forma simples e rápida procure pelo filme desejado e acesse sua página


Com a página aberta procure pelo Id do filme composto por uma sequência de números que precedem o título do filme na URL do site. Por exemplo:

https://www.themoviedb.org/movie/844-2046

Uma vez com acesso a esse Id, nosso sistema será capaz de obter automaticamente informações sobre o filme escolhido bastando apenas que você escreva uma descrição para ele.
